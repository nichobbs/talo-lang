# Talo collision checker — cross-language obscenity / slur blocklist
#
# Purpose (docs/0000 §4, docs/0001 §8): screen candidate Talo forms against
# strings that are obscene, taboo or slurs in major languages, so the lexicon
# never accidentally mints one. A frequently-skipped localisation step for any
# worldlang.
#
# Matching (tools/collision-checker `obscenityHit`): a candidate is FLAGGED (for
# human review, not silently banned) if its lowercased form
#   - exactly equals an entry, OR
#   - contains an entry of length >= 4 as a substring.
# Entries shorter than 4 characters therefore match ONLY as a whole word, so a
# short entry like `kut` cannot flag the legitimate root `kuto` (knife), and
# `sit` cannot flag `sita` (below) or `sitio` (website).
#
# CURATION
# - Entries are written in **Talo orthography** (the 20 letters
#   a e i o u p t k b d g c f s h m n l w y), because only such a string can ever
#   appear inside a Talo form. Foreign profanities are adapted the way the form
#   would actually enter Talo (r→l, v→w, z→s, j→y, sh→s; illegal codas/clusters
#   resolved), e.g. Korean *ssibal* → `siba`, Japanese *chinko* → `cinko`. A word
#   that survives no legal Talo adaptation (needs a cluster/coda Talo forbids) is
#   omitted — no Talo form could ever contain it, so it cannot match.
# - Verified against the live lexicon + derived layer: **no legitimate Talo form
#   is flagged** (the screen stays exit 0 over `data/lexicon.tsv`).
# - Broad, not exhaustive; a native-speaker pass per major L1 should still review
#   it before a v1.0 freeze. One entry per line; '#' = comment.
