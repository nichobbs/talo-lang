# Chapter 8 — The question-and-answer grid

English has a messy pile of words like *who, someone, no one, everyone, where,
nowhere, always, never*. Talo organises all of them into one tidy grid you can
generate yourself. Learn six beginnings and seven endings, and you get 42 words.

## Dialogue

```talo
A:  Bala datanto.        › Someone is coming.
B:  Sela?                › Who?
A:  Mi tauto hako.       › I know nothing.
B:  Ola tauto ituko.     › Everyone knows that.
```

## Grammar: stems × categories

Each of these words is a **beginning** (which kind) plus an **ending** (what it
refers to):

**Beginnings (stems):**

| stem | meaning |
|---|---|
| `se-` | question (which / wh-) |
| `ini-` | this, near |
| `itu-` | that, far |
| `ba-` | some |
| `ha-` | no, none |
| `o-` | every, all |

**Endings (categories):**

| ending | refers to |
|---|---|
| `-la` | person |
| `-ko` | thing |
| `-lo` | place |
| `-no` | time |
| `-fu` | reason |
| `-wa` | way / manner |
| `-mu` | amount |

Snap a beginning onto an ending and you get the word:

| | person -la | thing -ko | place -lo | time -no |
|---|---|---|---|---|
| **se-** (Q) | `sela` who | `seko` what | `selo` where | `seno` when |
| **ini-** (this) | `inila` this one | `iniko` this thing | `inilo` here | `inino` now |
| **itu-** (that) | `itula` that one | `ituko` that thing | `itulo` there | `ituno` then |
| **ba-** (some) | `bala` someone | `bako` something | `balo` somewhere | `bano` sometime |
| **ha-** (none) | `hala` no one | `hako` nothing | `halo` nowhere | `hano` never |
| **o-** (every) | `ola` everyone | `oko` everything | `olo` everywhere | `ono` always |

The same pattern continues for reason (`-fu`), way (`-wa`) and amount (`-mu`):
`sefu` why, `sewa` how, `semu` how much; `bawa` somehow, `hawa` no way, and so on.

## Using them

These words behave like the nouns or pronouns they stand for — they go in the
normal slot, no badge needed:

```talo
Hala manuto.         › No one eats.
Mi kanto bala.       › I see someone.
Te ikuto balo.       › He goes somewhere.
Ola tauto ituko.     › Everyone knows that.
```

Because the grid is perfectly regular, you never memorise these as separate
vocabulary. Need "nowhere"? It is just *none* + *place* = `halo`. Need "always"?
*every* + *time* = `ono`.

## Vocabulary

You already have it — six stems and seven endings generate the whole table. The
only ones worth fixing in memory as everyday words are: `inino` now, `inilo` here,
`itulo` there, `ono` always, `hano` never.
