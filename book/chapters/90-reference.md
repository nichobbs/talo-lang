# Grammar reference (one page)

Everything in this book, on a page.

## Word shape

```
content word = ROOT (+ piece)* + BADGE
BADGES:  -ka noun   -to verb   -pe modifier      (badge is always last)
```

## The pieces (derivation)

```
-ki agent   -tu tool    -bo result   -de place   -pa quality
-ci little  -go big     -ku opposite -ta cause    -pi become
compound:  describing-root + main-root + ONE badge
```

## Function words (no badge)

| | |
|---|---|
| pronouns | `mi` I · `yu` you · `te` he/she/it/they (plural with `pu`) |
| be | `yato` X is Y / X is (quality) · `kunato` there is / is located |
| role markers (after noun) | `na` to · `lo` at · `su` toward · `fe` from · `wa` with · `we` of |
| aspect (after verb) | `li` done · `wi` ongoing |
| number | `pu` plural · numbers after the noun |
| negation | `ne` before what it negates |
| questions | `ke` (yes/no, at end) · question words in place |
| joining | `i` and · `o` or · `ma` but · `fi` if · `toki` then |
| politeness | `ahoi` hi · `cao` bye · `doso` please · `tanku` thanks · `pole` sorry · `oke` okay |

## Sentence rules

```
1. Subject comes first.
2. The verb is found by its -to badge (otherwise word order is free).
3. A describing modifier (-pe) comes BEFORE its noun;
   a determiner (this/that, a number, a quantity) comes AFTER its noun.
4. Role markers and aspect/number particles follow the word they attach to.
5. The yes/no word 'ke' is always last.
```

## The correlatives grid

Stems `se-` (Q) · `ini-` this · `itu-` that · `ba-` some · `ha-` none · `o-` every,
times endings `-la` person · `-ko` thing · `-lo` place · `-no` time · `-fu` reason ·
`-wa` way · `-mu` amount. (E.g. `hano` = none+time = "never".)

## Numbers

`so` 0 · `ta` 1 · `ki` 2 · `mo` 3 · `fu` 4 · `le` 5 · `pikae` 6 · `haba` 7 ·
`cewa` 8 · `huba` 9 · `diko` 10 · `samu` 100.
