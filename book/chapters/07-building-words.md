# Chapter 7 — Building new words (the superpower)

This is the chapter that makes Talo small to learn but big to use. Instead of a
separate word for every idea, Talo lets you **build** words from roots you already
know, using a handful of regular pieces. Learn the pieces once, and thousands of
words open up.

## Dialogue

```talo
A:  Itula yato edukika.            › That person is a teacher.
B:  Te edukito belayadeka lo.      › He teaches at the school.
A:  Mi yato belayakika.            › I am a learner (student).
```

## Grammar: derivation affixes

Between a root and its badge you can slot in a small piece that changes the
meaning in a regular way. Here are the ten core ones:

| piece | meaning | example |
|---|---|---|
| `-ki` | one who does X (agent) | `edu`+`ki`+`ka` → `edukika` teacher |
| `-tu` | tool for X | `seka`+`tu`+`ka` → `sekatuka` a cutter |
| `-bo` | result of X | `edu`+`bo`+`ka` → `eduboka` a lesson |
| `-de` | place for X | `belaya`+`de`+`ka` → `belayadeka` school |
| `-pa` | quality of X | `gande`+`pa`+`ka` → `gandepaka` bigness |
| `-ci` | little X | `kanu`+`ci`+`ka` → `kanucika` puppy |
| `-go` | big X | `kanu`+`go`+`ka` → `kanugoka` big hound |
| `-ku` | opposite of X | `bone`+`ku`+`pe` → `bonekupe` bad |
| `-ta` | cause to X | `gande`+`ta`+`to` → `gandetato` to enlarge |
| `-pi` | become X | `gande`+`pi`+`to` → `gandepito` to grow |

The order is always **root → piece → badge**. The badge stays last, so you can
still read the part of speech off the end.

```talo
Edukika edukito belayakika.   › The teacher teaches the learner.
```

(From one root `edu` you get teacher, to-teach, lesson…; from `belaya` "learn" you
get learner and school.)

## The opposite-maker -ku

`-ku` is worth special attention: it builds antonyms for free. You never need a
separate word for "bad" — it is just "good + opposite":

```talo
# bone (good) → bonepe "good", bonekupe "bad"
# gande (big) → gandepe "big", gandekupe "small-ish / un-big"
```

## Grammar: compounds

You can also join two roots into one word. The describing root comes first, the
main root last, and one badge covers the whole thing:

```talo
# wato (water) + baiti (house) → watobaitika  "bathroom" (a water-house)
# piko (small) + hon (book)    → pikohonka     "booklet"
```

This is the same "describe before, the main thing after" order as everywhere else
in Talo.

## Why this matters

Because of derivation, the roughly 1,500 Talo roots cover far more than 1,500
ideas. When you meet a new compound or derived word, you can usually work out its
meaning from the pieces — and when you need a word you do not know, you can often
build a transparent one that others will understand.

## Vocabulary

| piece | meaning |
|---|---|
| -ki -tu -bo -de -pa | agent / tool / result / place / quality |
| -ci -go -ku -ta -pi | little / big / opposite / cause / become |
| belaya | learn |
| bone | good |
| hon | book |
