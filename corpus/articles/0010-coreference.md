# Coreference — self and each other

- **Register:** grammar exercise (not news) — the smallest text that puts the
  `0009` coreference pro-forms into running clauses.
- **Source:** original; constructed to exercise `0002` §6.8.
- **Proper nouns:** none
- **Exercises:** reflexive `sendi` (`te kanto sendi`), emphatic `sendi`
  (`mi sendi tendato`), reciprocal `salin` (`... pu tolonato salin`). All content
  words are real lexicon roots; `sendi`/`salin` are closed-class function words
  (`0002` Appendix B), so the gate accepts them without a lexicon row.

## English

The child washes itself. I did that myself. The friends help each other.

## Talo

```talo
# 1 — reflexive sendi, bound to the subject (0002 §6.8); oha "wash" (ACT-027)
totoka ohato sendi                  › The child washes itself.
# gloss: child-N wash-V REFL

# 2 — emphatic sendi, after the pronoun it intensifies; tenda "do" (ACT-001); itu "that"
mi sendi tendato itu                › I did that myself.
# gloss: I EMPH do-V that

# 3 — reciprocal salin over a plural subject, object slot; tolona "help" (ACT-037)
tomoka pu tolonato salin            › The friends help each other.
# gloss: friend-N PLURAL help-V RECIP
```
